ProyectoFinal


Egibide.fit se trata de una aplicación que monitoriza los pasos del usuario en tiempo real.
 
La aplicación guarda el número de pasos de cada usuario y realiza un ranking con el total de usuarios que están conectados. Muestra la posición del usuario respecto al grupo. 

También dispone de una opción para actualizar la clasificación que será útil en caso de que la aplicación este iniciada, pero no estemos realizando ninguna tarea. 

Una vez instalada la aplicación se necesita asociar una cuenta de Google y permitir el monitoreo de pasos de Google Fit. Dichos permisos solo se pedirán la primera vez que se instale la aplicación por lo que si se quiere cambar de cuenta se tendrá que desinstalar la aplicación y volver a instalarla.
